Site web fonctionnant sous php, avec les extensions PDO_MYSQL et GD2.
Nécessite une base de donnée MySql

Testé avec un serveur apache 2.4.9, php 5.5.12 et MySql 5.6.17.

Il faut d'abord créer la base de donnée avec base.sql puis y injecter des données dedans avec donnees.sql.
De base le login pour la base de donnée est 'root' et il n'y a pas de mot de passe. Ces informations se trouve dans Database.php.

La page de demarrage du site est index.php.
